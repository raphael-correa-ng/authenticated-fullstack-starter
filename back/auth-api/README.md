## AuthApi

Maven API for Auth service

##### Install

<pre>
mvn clean install
</pre>

##### Then include this dependency in your project's pom

```xml
<dependency>
    <groupId>com.rcs</groupId>
    <artifactId>fullstack-starter-auth-api</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
